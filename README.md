# ✦ AI Study Buddy

An AI-powered study companion that explains complex topics, summarizes notes, and generates quizzes — built with Python, Streamlit, and Google Gemini AI.

## 🌐 Live Demo
👉 [vishv-ai-study-buddy.streamlit.app](https://vishv-ai-study-buddy.streamlit.app)

## ✨ Features
- 💡 **Explain Topic** — Get clear explanations at beginner, intermediate, or advanced level
- 📝 **Summarize Notes** — Get bullet points, paragraph summary, or key concepts
- 🧠 **Quiz / Flashcards** — Generate interactive flashcards from any topic

## 🛠 Tech Stack
- Python + Streamlit
- Google Gemini AI
